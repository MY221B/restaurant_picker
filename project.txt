# 餐厅随机推荐系统

这是一个基于Web的餐厅随机推荐系统，可以根据用户位置和偏好推荐附近的餐厅。

## 项目结构

### 前端文件
- `index.html`: 主页面，包含用户界面元素
  - 位置输入框
  - 城市选择下拉框
  - 距离滑块
  - 餐厅展示区域
  - 历史记录区域

- `script.js`: 主要的前端逻辑
  - 处理用户交互
  - 加载和处理餐厅数据
  - 随机选择餐厅
  - 计算和显示距离信息
  - 管理历史记录

- `styles.css`: 基础样式文件
- `output.css`: 由Tailwind编译生成的样式文件

### 数据处理
- `rest_data_process/restaurants_lat_long.py`: 餐厅地理信息处理脚本
  - 读取餐厅CSV数据
  - 调用高德地图API获取经纬度
  - 使用多种方式尝试地址匹配
  - 更新CSV文件中的经纬度信息

- `distance.js`: Node.js脚本，用于批量计算距离
  - 计算用户位置到餐厅的驾车距离和时间
  - 支持批量处理以避免API限制
  - 生成包含距离信息的新CSV文件

### Web Worker
- `distanceWorker.js`: 后台线程脚本
  - 使用哈弗辛公式计算直线距离
  - 在不阻塞主线程的情况下处理距离计算
  - 支持实时距离筛选

## 主要功能
1. 餐厅数据管理
   - CSV格式存储餐厅信息
   - 自动获取和更新地理坐标
   - 支持多城市数据

2. 位置服务
   - 用户位置输入和更新
   - 城市筛选
   - 距离计算（直线距离和驾车距离）

3. 推荐系统
   - 随机推荐算法
   - 距离和时间筛选
   - 历史记录追踪

4. 用户界面
   - 响应式设计
   - 交互式地图（计划中）
   - 历史记录展示

## 数据格式
restaurants.csv 包含以下字段：
- 名称：餐厅名称
- 地址：详细地址
- city：所在城市
- 经纬度：地理坐标
- tel：联系电话
- time：收藏时间
- 其他距离字段：到不同位置的距离信息

## 使用的API
- 高德地图API
  - 地理编码服务
  - 驾车路径规划
  - 步行路径规划（计划中）

## 开发工具
- Tailwind CSS：样式框架
- Papa Parse：CSV解析
- Web Workers API：后台计算
- 高德地图JavaScript API

## 如何运行
1. 处理新餐厅数据：
   ```bash
   python rest_data_process/restaurants_lat_long.py
   ```

2. 计算距离信息：
   ```bash
   node distance.js
   ```

3. 启动Web服务：
   使用任意HTTP服务器托管项目文件

4. 编译CSS：
   ```bash
   npx tailwindcss -i ./styles.css -o ./output.css --watch
   ```
